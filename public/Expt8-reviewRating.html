About the experiment

As it would be easy to interpret the rating when rendered as an image like stars. This experiment is about rendering equal number of stars for the user's rating on a product. A new directive called 'starRating' has been added which can be used like any other directive in the html. The directive iterates over a loop for 5 times(maximum rating is 5) rendering a star using the ascii value. The angular js code to render stars is as follows :

Code Snippet:

app.directive('starRating', function () {
return {
restrict: 'A',
template: '<ul class="rating">' +
'<li ng-repeat="star in stars" ng-class="star">' +
'\u2605' +
'</li>' +
'</ul>',
scope: {
ratingValue: '=',
max: '='
},
link: function (scope, elem, attrs) {
scope.stars = [];
for (var i = 0; i < scope.max; i++) {
scope.stars.push({
filled: i < scope.ratingValue
});
}
}
}
});

The 'starRating' directive is then used in the html as follows :

Code Snippet:

<div star-rating rating-value="review.rating" max=5 ></div>
