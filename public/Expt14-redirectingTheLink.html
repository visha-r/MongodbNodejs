About the experiment

This experiment is about redirecting user to the right tab based on the activity he carries out. This is a very important feature as this makes the application very user friendly. Everytime the user adds a product to his cart or favorites list, he should be given some hint that his activity was successful showing the status of the transaction he just made. This experiment is focused on doing that. So whenever the user adds a product to his cart or favorite, he wil be redirected to the corresponding tab in the account page. I maintain a variable for each tab which will be passed along while redirecting the user to his profile page. The variable value depends on the activity he carries out. In the html, I use this value to make the tabs active.
For example, on adding a product to the cart, I make the direction as follows :
Code Snippet:

ProductService.addToCart($scope.user);
$location.url('/profile/'+"cart");
Here the value cart in the url says that, the user has to be redirected to the cart tab in the profile page. In the controller I add the active class to the tab based on the incoming value as follows :
Code Snippet:

var tab = $routeParams.activeTab;
if(tab == 'favorite'){
$scope.favActive = true;
}
else if(tab == 'cart'){
$scope.cartActive = true;
}
else if(tab == 'review'){
$scope.reviewActive = true;
}
else if(tab == 'network'){
$scope.networkActive = true;
}
else {
$scope.userActive = true;
}
