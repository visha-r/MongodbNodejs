About the experiment

The experiment is focused on designing the product details page where the user can see all the features of the given product along with its specification. The page also displays the amount the user will save on buying a particular product.The page has 2 tabs one of which lists the specifications and the other product features. The user also has an option to add the current product to his cart. But then the user will be forced to login if he tries to add a product without logging in. Otherwise upon adding the product to the cart successfully, the user will be redirected to his profile where he can see his favorites and cart.

Bootstrap.ui tabs has been used to display the tab. ui.bootstrap is added as a dependency in the controller as follows :

Code Snippet:

var app = angular.module('AngularApp', ['ngRoute', 'ui.bootstrap']);
